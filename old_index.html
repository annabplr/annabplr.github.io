pozi

une application web qui permet de prévoir son menu de la semaine à la maison.
cette application web a été codé en ruby on rails, avec du javascript (ajax notamment). elle est disponible sur mobile via une Progressive Web App. l'une de ses principales fonctionnalités est de générer un menu pour la semaine, une to-do list et une liste de courses en fonction des choix faits initialement.
